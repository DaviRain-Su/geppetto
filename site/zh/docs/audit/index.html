Audit 与 Review

geppetto audit 当前是基础静态 review gate，目标是小而确定。

默认检查

cargo fmt --check
cargo check --all-features

Strict 检查

cargo clippy --all-targets --all-features -- -D warnings

Review 知识

Rust crate 还在 src/anti_patterns.rs 中包含 review guidance，并在 guard、schema、dispatch 中提供可执行约束。

当前限制

它还不是完整 PR code review 产品。下一步是结构化 finding、report 输出和可执行 anti-pattern 检查。
