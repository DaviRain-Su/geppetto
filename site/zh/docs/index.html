Geppetto 中文文档

Geppetto 是面向 Solana builder 和 AI code agent 的工程工作流。它把 Pinocchio 知识 SDK 与稳定 CLI 工作流契约结合起来。

当前形态：SDK + agent guidance + init、new、test、audit、deploy CLI 命令。

从这里开始

  • 快速开始：安装和运行本地 CLI。
  • CLI 参考：完整命令面。
  • 部署：Platform workflow contract。
  • SDK 概念：guard、schema、dispatch 和知识模块。

产品主张

Geppetto 不替代 Solana 工具链，而是让 repo 自身更容易被人和 AI Agent 理解、验证和继续开发。
