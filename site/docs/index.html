Geppetto Docs

Geppetto is an agent-native engineering workflow for Solana builders. It combines a Pinocchio-focused Rust knowledge SDK with a CLI that exposes stable workflow contracts.

Current shape: SDK + agent guidance + CLI commands for init, new, test, audit, and deploy.

Start here

  • Quickstart for installing and running the local CLI.
  • CLI Reference for every command surface.
  • Deploy for the platform workflow contract.
  • SDK Concepts for guard, schema, dispatch, and knowledge modules.

Product thesis

Geppetto does not try to replace Solana tooling. It makes the repository itself easier for humans and AI agents to understand, validate, and continue.
